Reliable Transport Protocol

Nov 2025
Python • Networking • Wireshark

Developed a TCP-like connection-oriented transport protocol from scratch using Python and UDP to establish reliable data transfer over inherently unreliable channels. The implementation uses a custom Go-Back-N (GBN) sliding window mechanism, utilizing MD5-based checksums for data integrity and a structured three-way handshake for robust connection management. The project involved building a dedicated channel layer to simulate real-world network conditions, including random packet loss and bit corruption.

Focused on high-performance congestion control by implementing a dynamic Congestion Window (CWND) that transitions between Slow Start and Congestion Avoidance phases. The system manages network stress by halving the slow-start threshold and resetting the CWND upon detecting loss via a retransmission timer. Validation through Wireshark traffic analysis confirmed that the protocol correctly adheres to the receiver’s advertised window and maintains fairness between simultaneous flows, remaining functional even under 20% aggregate packet loss.
